Abaixo, indico alguns projetos que desenvolvi com Unity, aplicando lógica de programação e linguagem C#

GameBird - um jogo estilo ao famoso game Flappy Bird, onde crio a ilusão de uma tela infinita, onde os objetos de colisão são gerados aleatóriamente gerando sempre um desafio novo cada vez o game é jogado!

Jogo de Nave-2D - Este foi um projeto desenvolvido com as aulas do professor Gabriel do canal Desenvolvendo Jogos no youtube. Seguindo o estilo tela infinita do game anterior, nesse projeto desenvolvi mais a fundo propriedades de lógica de programação e orientação a objetos como herança, polimorphismo e utilidades mais avançadas da plataforma Unity, como os Prefabs.
